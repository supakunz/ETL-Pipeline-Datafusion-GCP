
# Book Sales End-To-End Data Engineering Project on Google Cloud Platform

An end-to-end modern data engineering project, including deployment of ETL pipeline on Google Cloud Platform, using BigQuery for data analysis and leveraging Looker to generate an insight dashboard.

## Technology Stack
Languages: 
* Python
* SQL

Google Cloud Platform: 
* Google Storage
* Cloud Data Fusion
* Google Composer
* Big Query
* Looker Studio


## Data Storage

The raw data files are stored in the `data` folder of the repository. These files are used as input for the processing pipeline and are essential for the overall project.
